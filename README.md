# Sistema de Mercearia

<p align="center">
Um sistema desenvolvido para facilitar o controle de estoque, vendas e a organização de uma mercearia.
</p>

---

## Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de auxiliar no gerenciamento de uma mercearia, proporcionando mais organização no controle de produtos, no acompanhamento do estoque e no registro das vendas.

---

## Funcionalidades

- Cadastro de produtos  
- Edição e remoção de informações  
- Controle de estoque  
- Registro de vendas  
- Organização de dados para relatórios  

---

## Tecnologias Utilizadas

- Java  
- MySQL  
- NetBeans   
- Git e GitHub  
- Linux
